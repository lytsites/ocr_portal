# Form Portal

Единый фронтенд для форм:

- `2-19`
- `2-43`
- `4-20`
- `5-52`

Основной публичный origin:

- `https://docs.e-qoldau.asia`

Основной API:

- `https://api.e-qoldau.asia`

## Локальный dev

```powershell
npm install
npm run dev
```

По умолчанию production-конфиг уже ориентирован на `api.e-qoldau.asia`.

## Маршруты

- `/2-19`
- `/2-43`
- `/4-20`
- `/5-52`
- `/analytics`

## Идея

Это один общий UI, который работает как единый портал на `docs.e-qoldau.asia`, а не набор отдельных localhost-фронтов.
